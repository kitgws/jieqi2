杰奇1.8安装使用和更新说明

一、1.8程序需要 php 5.3.x + zend guard loader + mysql 5.x

二、安装程序还没整理，请手工导入install目录下的 struct.sql 和 data.sql，然后修改 /configs/define.php 设置正确数据库参数后登陆后台管理

三、代理原创作品功能
    代理模式为：由杰奇书盟(book.jieqi.com)获取原创站共享的作品信息，加盟站再通过书盟接口获取作品到本站。其中免费章节内容可在本地保存，vip章节可以在本站显示，但是内容部分通过接口远程调用原创站图片显示，用户充值也通过书盟接口实现。
    
    代理模式配置文件：/configs/setting.php ，主要是网站id，通讯密钥，和对应类别等设置，可以根据实际情况手工编辑。
    
    同步作品：后台小说连载模块->同步杰奇书盟。本程序默认同步书盟所有作品，如果一次没有全部同步完成关闭了浏览器，下一次执行本程序会继续同步。但是不要开多个窗口同时执行本程序，否则可能内容错乱。

四、目录页的章节名数组改变
1.7以前版本章节数组用 $indexrows，预先设置好一行几列，显示成
{?$indexrows[i].cname1?} {?$indexrows[i].cname2?} {?$indexrows[i].cname3?}

1.8开始，暂时保留之前写法，但是推荐新的写法，所有章节在 $chapterrows，不预先赋值 cname1 cname2，由css控制一行几列，或者模板循环里面加判断控制
